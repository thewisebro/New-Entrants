DC++ Help

Getting started with DC++

 Download

1. Download

Click on the Download link to get the DC++ setup.

2. Install

Run the setup executable file to begin installation. It is highly recommended that you leave all the items in the Installation Options screen selected.

3. Settings

There are a few basic settings that need to be entered before you can use DC++. You can open up the Settings dialog from File menu.

For now, leave them as it is.

4. Share Files

Sharing is essential to enable you to get into hubs and download anything. The minimum requirement for sharing is 2GB of data and a minimum of 3 slots on the IMG DC++ Hub.

Go to Settings → Sharing → Shared directories. You can add or remove shared directories in this page.

To add files to share, click on the Add folder button and select a folder that contains files you wish to share. You will be prompted to add a virtual file name.
Once you click on OK, the hashing progress window will appear.

4. Connect to the IMG DC++ Hub

Open up DC++ and go to File → Quick Connect.

Type in dc.channeli.in and click OK to connect.

Connected!

5. Browsing Files

To browse a user's file list, select a user from the list in the hub window and right click on their name. Select "Get file list" and the entire file list should start downloading.

Once the list has downloaded it will automatically open a new window with the user's name on the tab at the top.

To queue up a file for download, either double click on the file or right click on it and select Download. Note that the entire directory and its subdirectories will be added to your queue.

5. Searching for Files

Press Ctrl + S

Enter the name of the file you wish to search for.

Once you click on Search, the window should show some results.
Select one or more of the results and right click on them. Select Download to queue them up.

Good luck and happy sharing!
