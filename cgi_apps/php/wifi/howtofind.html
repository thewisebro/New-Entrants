MAC Address

A Media Access Control (MAC) address is a unique identifier assigned to the network adapters or network interface cards (NICs) by the manufacturer for identification pupose.It may also be known as an Ethernet Hardware Address (EHA), hardware address, adapter address, or physical address.

To find the MAC address of your machine, corresponding to Ethernet adapter:

For Windows users:

  • Start the command line. (Windows+R, and then type 'cmd' to run command line)
  • Type ' ipconfig /all ' in the command line and press enter.
  • Certain details will appear on your computer. Among these you will be able to find the MAC address called Physical address of your Ethernet adapter.Please note that this number is unique to each and every adapter/computer. Thus, please be careful while checking and submitting this address, as you won't be able to access the internet without a correct MAC address.

For Linux Users:

  • Start the command line.
  • Type ' /sbin/ifconfig | grep HWaddr ' in the command line and press enter.
  • Certain details will appear on your screen. Among these you will be able to find the MAC address called Physical address of your Ethernet adapter(Usually eth1 or Wlan0).Note that, generally eth0 is corresponding to wired lan adapter.

Please note that this number is unique to each and every adapter/computer. Thus, please be careful while checking and submitting this address, as you won't be able to access the internet without a correct MAC address.
